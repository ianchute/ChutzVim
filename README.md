ChutzVim
========

Install vim with the most popular plug-ins 
in THREE easy steps in the terminal!

1. download and unzip ChutzVim to $HOME
2. execute: chmod 777 ChutzVim.sh
3. execute: ./ChutzVim.sh

read _vimrc file for details
